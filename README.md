## Si /admin no abre o no loguea
1) Netlify → Site settings → **Identity** → **Enable**
2) Identity → Registration: “Invite only”
3) Identity → Invite users → invita `gc.asin.zapata@gmail.com`
4) Acepta la invitación desde el correo
5) Entra a `/admin/` y presiona “Ingresar”
